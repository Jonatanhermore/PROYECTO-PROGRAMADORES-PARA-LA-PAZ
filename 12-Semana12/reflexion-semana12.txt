Durante este curso aprendí que la tecnología puede convertirse en una herramienta importante para fortalecer la participación ciudadana y la organización comunitaria. También comprendí la importancia de utilizar bases de datos para almacenar información de forma organizada y segura. Aprendí cómo proteger el acceso mediante autenticación de usuarios y la necesidad de cuidar la privacidad de los datos personales. Además, conocí el uso de Git y GitHub para controlar versiones y organizar proyectos. Igualmente entendí cómo la automatización con herramientas como n8n y Telegram puede facilitar la comunicación y el seguimiento de actividades. Finalmente, comprendí que una solución tecnológica no solo debe funcionar correctamente, sino que también debe ser segura, transparente y útil para la comunidad.
