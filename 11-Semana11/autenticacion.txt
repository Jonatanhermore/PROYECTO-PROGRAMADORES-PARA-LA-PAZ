La autenticación o login permite identificar a un usuario antes de darle acceso a una aplicación. Gracias a este proceso, solo las personas autorizadas pueden utilizar ciertas funciones del sistema. Algunas rutas deben estar protegidas porque contienen información privada o permiten realizar cambios importantes. De esta manera se evita el acceso no autorizado. No deben exponerse públicamente contraseñas, documentos de identidad, teléfonos, direcciones, tokens, claves de API ni otros datos sensibles de los usuarios.
