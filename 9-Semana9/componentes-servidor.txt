Sistema operativo:
Es el software principal del computador. Administra los recursos del sistema y permite ejecutar programas y aplicaciones.

Node.js:
Es un entorno de ejecución que permite usar JavaScript fuera del navegador para crear aplicaciones del lado del servidor.

Aplicación server.js:
Es el archivo que contiene el código del servidor. Define las rutas y respuestas que recibirán los usuarios.

Puerto:
Es un punto de comunicación utilizado por el servidor para recibir solicitudes. En esta práctica se utiliza el puerto 3000.

Logs:
Son registros que almacenan información sobre eventos y actividades del sistema. Permiten monitorear el funcionamiento y detectar errores.
